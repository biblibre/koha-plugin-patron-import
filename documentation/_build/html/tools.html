Outils

Cronjob

Le cronjob doit être lancé de manière journalière, de préférence la nuit pour éviter les interférences avec l’usage normal en parallèle de Koha.

Exemple:

PERL5LIB=/path/to/koha
KOHA_CONF=/path/to/koha-conf.xml
PATH_TO_PLUGIN=/path/to/plugin

0 23 * * * perl $PATH_TO_PLUGIN/Koha/Plugin/Com/Biblibre/PatronImport/cron/run-import.pl -i 1 -f /path/to/patrons.csv

Lanceur

Un lanceur est disponible dans les sources du plugin pour vous aider à lancer vos imports automatiquement (ex crontab).

Disponible ici $CHEMIN_DU_PLUGIN/Koha/Plugin/Com/Biblibre/PatronImport/bin/launcher.sh
