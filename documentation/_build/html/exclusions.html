Exclusions

Les règles d’exclusion peuvent être utilisées pour définir les adhérents à ignorer.

Une règle est un ensemble champ koha => valeur. Toutes les règles doivent correspondre pour que le patron soit ignorés.

Il y a deux types de règles:

  • Externe: correspond à la donnée entrante

  • Koha: correspond à la donnée Koha (si le patron correspondant à été trouvé).

Créer une règle

Entrer le nom d’une règle, cocher ou non s’il s’agit d’une règle Koha et cliquer sur le bouton “Add”.

Ajouter les champs

Sélectionner un champ, entrer une valeur et cliquer sur le bouton “Add”.
