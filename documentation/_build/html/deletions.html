Suppressions

Les règles de suppressions sont un ensemble de champs/valeurs qui définit les patrons à supprimer de Koha.

Plusieurs conditions doivent être réunies pour supprimer un adhérent:

  1. La règle doit correspondre en entier aux données entrantes

  2. Les données entrantes doivent être associées à un adhérent Koha (trouvé par dédoublonnage)

  3. L’adhérent cible ne doit pas avoir de dettes

  4. L’adhérent cible ne doit pas avoir de prêts dus

Créer une règle

Saisir un nom de règle et cliquer sur le bouton « Add ».

Ajouter les champs

Sélectionner un champ, saisir une valeur et cliquer sur le bouton « Add ».
