Attributs étendus

Cette page est utilisée pour configurer le comportement de chaque attribut étendu d’un adhérent Koha existant lors de sa mise à jour.

Pour chacun des attributs, le comportement peut être configuré de différentes façons:

Note

Si l’attribut est non répétable, le menu de sélection correspondant à « If many values » sera désactivé.

L’attribut est déjà défini avec une valeur (seulement une)

  • Do nothing: Rien ne sera ajouté ni mis à jour

  • Update existing: remplacement de la valeur existante par la valeur entrante

L’attribut est déjà défini avec plus d’une valeur (plusieurs)

  • Do nothing: Rien ne sera ajouté ni mis à jour

  • Update existing: remplacement de la valeur existante par la valeur entrante

  • Add if the values does not exists: ajoute la nouvelle valeur entrante seulement si elle n’existe pas déjà.

Important

Lorsque « Update existing » est sélectionné, vérifier que la donnée entrante corresponde avec la donnée que vous souhaitez remplacer, même si la donnée est vide ou contient une chaîne vide.
